# Data Engineering Pipeline with Hadoop

## Data engineering project with web scraping, Hadoop, and PostgreSQL

---

### Overview
This project demonstrates an end-to-end data engineering pipeline that collects, processes, stores, and analyzes data from multiple sources.

---

### Key Features
- **Web Scraping** using Selenium and BeautifulSoup  
- **API Integration** (Yahoo Finance)  
- **Distributed Storage** with Hadoop HDFS  
- **Database Management** using PostgreSQL  
- **Automated Data Ingestion Pipeline**

---

### Tech Stack
- **Python**
- **Selenium**
- **BeautifulSoup**
- **PostgreSQL**
- **Hadoop (HDFS)**
- **Docker**

---

### Outcome
This project showcases practical implementation of **data pipelines**, **big data storage**, and **data processing workflows**.
